# hubot-jira

A hubot script that does the things

See [`src/jira.coffee`](src/jira.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-jira --save`

Then add **hubot-jira** to your `external-scripts.json`:

```json
[
  "hubot-jira"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
